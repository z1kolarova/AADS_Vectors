Detailed Description

This module defines the API interface to the Vector data structure. The data structure is designed to contain numerical data.

◆ Vector_Create()

Vector_t* Vector_Create ( size_t  initial_size,
size_t  alloc_step 
)

Creates a vector with initial_size and a alloc_step.

The returned pointer points to the heap where is allocated new instance of the Vector_t structure. After return from the function the owner of a pointer to a vector structure is the user instead of the library and user is responsible for freeing the memory.

Parameters
[in]initial_sizeInitial items count of the vector.
[in]alloc_stepNumber of items that are allocated to the vector when run out of memory.
Returns
Pointer to the allocated vector structure or NULL in case of failure.

◆ Vector_Fill()

void Vector_Fill ( const Vector_t *const  vector,
Vector_DataType_t  value,
size_t  start_position,
size_t  end_position 
)

Fills a portion of vector specified by a range with a desired value.

Vector is overwritten from the start_position to the end_position (including it). If the end_position is located out of vector boundaries, vector is filled from the start_position to the last item of the vector. If the start_position is located behind the last item of a vector, no changes are made.

Parameters
[in]vectorPointer to a vector.
[in]valueValue to be set.
[in]start_positionStarting position.
[in]end_positionEnd position.
